apkgo

一行命令，将 APK 发布到所有主流安卓应用商店

GitHub Download
apkgo
# 生成配置
$ apkgo init --store huawei,xiaomi,oppo

# 一键上传到所有商店
$ apkgo upload -f app-release.apk --notes-file CHANGELOG.md
